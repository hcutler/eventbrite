Setup

1. set up a virtual environment in Python3 and install packages	in requirements.txt

2. run ‘python manage.py migrate’ to initialize the database

3. run ‘python manage.py sync_categories’ to pull category information from the API and write to you local database

4. rum ‘python manage.py run server’ to run the server

5. open your browser and go to ‘http://127.0.0.1:8000/events/'
